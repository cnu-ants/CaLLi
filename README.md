# CaLLi

  CaLLi is a OCaml Analysis Library for LLVM IR

## Requirements

  * OCaml, version 4.13.1 or later.  

## Installation

1) Using OPAM
```
  opam install calli
```

2) Building from source
```
  dune build
  opam install .
```

## Usage

Read our [Getting Started](https://github.com/cnu-ants/CaLLi/blob/main/example/README.md) page for details.
